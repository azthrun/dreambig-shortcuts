# DreamBig Shortcuts

A collection of utility libraries and shortcuts for .NET development, designed to simplify common programming patterns and improve code quality.

## Projects

### 🎯 [DreamBig.Shortcuts.Results](./DreamBig.Shortcuts.Results/)

[![NuGet](https://img.shields.io/nuget/v/DreamBig.Shortcuts.Results.svg)](https://www.nuget.org/packages/DreamBig.Shortcuts.Results/)

A lightweight Result pattern implementation for handling success and failure outcomes in .NET applications. Provides type-safe error handling with support for error messages, exceptions, and HTTP status codes.

**Key Features:**
- Type-safe Result<T> pattern
- Flexible error information (messages, exceptions, or both)
- HTTP status code support for web APIs
- Railway-oriented programming support
- Zero dependencies

[📖 Read the full documentation →](./DreamBig.Shortcuts.Results/README.md)

**Installation:**
```bash
dotnet add package DreamBig.Shortcuts.Results
```
